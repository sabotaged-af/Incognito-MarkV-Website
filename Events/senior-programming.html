Event Details

  • Classes: XI-XII
  • No. of Particpants per Team: 1
  • No. of Rounds: 2
  • Programming Language: Python
  • Finals: 6 Teams will Advance
  • Event Head: Samyak Jain
  • Submission form/email : www.example.com / example@gmail.com

General Guidelines


  • The contest will be hosted on HackerRank

  • PRELIMS will be held on 18th OCTOBER

  • The contest comprises of 10 questions for PRELIMS round and 6 questions for the FINALS with varying difficulty levels.

  • All the necessary instructions and points will be specified along with the question.

  • Each question has points allocated to it, get all the test cases right to score maximum points.

  • Top 6 teams will qualify for the FINALS which would be held on 23rd OCTOBER.

  • The Programming question must be answered using Python with the help of the on-screen editor provided on the system. Avoid creating source code on your computer and uploading on the website.

  • Each team can submit the code as many times, however only the last submitted code will be considered for evaluation.

  • In case the participants are caught copying from the internet or cheating by any means, it would lead to immediate disqualification of the team.

  • Prelims Results will be posted on the website.

Judgment Criteria


  • Creativity

  • Completion

  • If in case two or more teams have the same points, the leader board will be adjusted according to the aggregate time taken by the teams to complete the challenges

  • Decision of the judges will be final and binding
